# vue-model-conversion

## 项目设置
```
npm install
```

### 开发的编译和热加载
```
npm run serve

端口修改地址：
* 所在目录：node_modules\@vue\cli-service\lib\commands
* 修改文件：serve.js
```

### 生产编译和打包
```
npm run build
```

### 运行测试
```
npm run test
```

### 提示和修复文件
```
npm run lint
```
